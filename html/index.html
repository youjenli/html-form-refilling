Chrome 和 Edge 回填表單的問題

當開發人員在瀏覽器端調整過表單元素的 action 後，Chrome 和 Edge 在使用者用 history.go(-1) 回到表單頁時不會代回先前輸入的值。

只要不動到表單的 action 屬性，那 Chrome 和 Edge 就會在使用者用 history.go(-1) 回到表單頁時代回先前輸入的值。

已確定有此特性的瀏覽器

  • Chrome 版本 96.0.4664.45 (64bit)
  • Edge 版本 95.0.1020.53 (64bit)

已確定無此特性的瀏覽器

  • Firefox 94.0.1 (64bit)
  • Windows 10 21H1 build 19043.1348 內建的 IE 11
